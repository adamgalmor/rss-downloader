A simple RSS link downloader written in Python. It can be used to download media files from RSS feeds, such as podcasts or video channels.

To run this project:
1. Install UV (a Python package manager) if you haven't already: https://uv.per.dev/
2. Clone this repository and navigate to the project directory.
3. Install the required dependencies using UV: `uv sync`
4. Update the `rss_url` variable in `main.py` with the URL of the RSS feed you want to download from.
5. Run the script: `uv run main.py`
